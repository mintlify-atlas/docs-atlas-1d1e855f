WebJET CMS is an enterprise-grade content management system built on the Spring Framework. It gives content editors a powerful visual editor, gives administrators fine-grained user and security controls, and gives developers a clean extension model for building custom applications — all in one platform trusted by major banking institutions for over 20 years.

## Getting started

<Steps>
  <Step title="Install WebJET CMS">
    Deploy the WAR file to Tomcat 10+ or run locally with Gradle. See the [Installation guide](/install/overview) for step-by-step instructions.
  </Step>
  <Step title="Configure your site">
    Set up your database connection, SMTP email, and site-specific settings via the [Configuration reference](/install/configuration).
  </Step>
  <Step title="Add your first page">
    Log into the administration, navigate to **Web Pages**, and create your first page using the built-in editor or Page Builder. See the [Editor guide](/editor/webpages).
  </Step>
  <Step title="Extend with applications">
    Embed ready-made applications like [News](/editor/apps/news), [Forms](/editor/apps/forms), or [E-commerce](/editor/apps/eshop) by inserting them into any page.
  </Step>
</Steps>
